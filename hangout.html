Introduction

Mobile application design project with the goal of optimizing event coordination experience and practicing Material Design.

I love to initiate events and invite friends to try out new stuff. However, it's never easy for me to do event coordination, since nowadays we simply have way too many ways to communicate with each other. Some of my friends would only reply to text, some of them only pay attention to Facebook, others use various apps as communication channel only. All these different tools has made indivisual communication so much easier than the early days, but, at the same time, it also has made group communication so much harder. I counldn't help but wonder: how can we make group communication experience as smooth as indivisual communication and furthur help with event coordination?


Tools

Pen and Paper, Adobe Illustrator and Photoshop.


Process
  • Define goal and scope of the project.
  • Research on user behaviors by interviews and conduct competitor analysis.
  • Ideate design options by pen and my Moleskine.
  • Build Lo-fi and High-Fi Prototypes to demonstrate/explore user flow, UI, interactions and visual treatment.
  • Critique and iterate on the design.
