Introduction

Windows Mobile Application Design. This project was completed in the spring 2013 for COM589: Interactive Design. We aimed to design a mobile application with the goal of increasing customers' shopping intention.


Responsibility

UX Designer and Prototyper delivered interactive prototypes by Axure, user flow and site map.


Design Process
  • Conduct research on user behavior by interviews and competitor analysis.
  • Ideate design options according to findings from the interviews.
  • Build interactive prototype by Axure to demonstrate UI and interaction of the application.
